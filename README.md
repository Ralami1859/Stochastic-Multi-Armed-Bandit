Implementation of 7 multi-amred bandit algorithmm for the stochastic Bernoulli environment:
<ul>
 <li>Thompson Sampling </li>
 <li>KL UCB </li>
 <li>Bayes UCB </li>
 <li>UCB 1 </li>
 <li>UCBV </li>
 <li>MOSS </li>
 <li>CPUCB </li>
 <ul>
